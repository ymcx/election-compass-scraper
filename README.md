# Election compass scraper

Scrapes candidates' responses from Yle's election compass (vaalikone). The results for each election are saved to a file in the data directory. By default, the latest election is chosen if it hasn't been specified in the arguments.

<br>

```shell
$ election-compass-scraper [OPTIONAL: election]
```
